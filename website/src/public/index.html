Interceptor is a browser extension that mocks AJAX requests at the browser level so you can work on the front end UI without really starting a backend server.

Install Interceptor from

Chrome Web Store Firefox Add-ons

Features

Using Interceptor, you can mock and/or modify HTTP AJAX requests.

Listen to Incoming Requests

Interceptor lists out all the requests requested by the browser on a per-tab basis. You can choose to enable/disable the Listen feature whenever you want to.

Modify the Response

You can inspect each request and modify the response as you need, and interceptor will make sure that the modified response will be sent to the browser tab when requested again.

Inspect Server Response

You can see the actual response sent out by the website's backend whenever you need to. It will help you know the structure of response which helps you modify it and intercept.

Add interception rules

You can add a interception rules for requests and define responses that haven't been recieved yet. So you can define a mock response for a request even before it comes back from the server.
